We have a parent element with class flex-container holding children items. Div is used to break up an HTML document into divisions, but is a common target to hold other things that can be styled with CSS. In this case we have two rows of 3 items in our container. FlexBox is primarily used to linearly layout items.

Direct child elements(s) of the flexible container automatically becomes flexible items.
